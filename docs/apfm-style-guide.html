Do / Don't — the "not Claudey" rules

Do: warm cream canvas, soft-serif headlines, coral for the one key action.
Do: navy structure, sage/amber/coral semantics, generous whitespace.
Do: real photography for hero/testimonial moments; plain human copy.
Don't: purple/indigo, gradient "AI" glows, ✨ sparkle motifs.
Don't: default Inter/Roboto headlines, everything-pill-shaped, glassmorphism.
Don't: stark white full-page bg, dark-mode-by-default, emoji as icons.

Palette is an APFM-aligned derivation (Ink #333333 and the Dove/Nile Blue names are confirmed; exact brand-guide hexes to be swapped in if available). Type pairing recommended to match the brand's empathetic/editorial voice. Fonts: Fraunces + Mulish (Google Fonts).
